# 로하이마켓 × 비움마켓 랜딩페이지

## 프로젝트 개요
**로하이마켓(Lohigh Market)**의 신뢰도(팔로워/커뮤니티 규모)와 **비움마켓(Bium Market)**의 혁신성을 동시에 강조하는 프리미엄 원단 소싱 랜딩페이지입니다.

- **컬러 팔레트:** 네이비(`#0a1628`), 슬레이트(`#2d3e50`), 골드(`#c9a84c`), 비움 틸(`#0d9488`), 화이트(`#f8f9fa`)
- **폰트:** Noto Sans KR, Inter, Playfair Display

---

## 구현된 섹션

| 섹션 | 내용 |
|------|------|
| **Hero** | 풀스크린 네이비 배경 + 패브릭 패턴, 카운터(16K/6K), 2개 CTA 버튼 |
| **About** | 브랜드 소개, 패브릭 카드 비주얼(린넨/코튼/데드스톡), 핵심 특징 리스트 |
| **Stats** | 4개 신뢰 지표 카드 (밴드 16,000+, 인스타 6,000+, 국내최다 재고, 10년 경험) |
| **Services** | 로하이마켓 몰 vs 비움마켓 2-컬럼 서비스 카드 |
| **Bium Launch** | 비움마켓 런칭 카운트다운, 이메일 사전등록, 파티클 배경 |
| **Brand Story** | 브랜드 비전 인용구 + 신뢰/혁신/지속가능성 가치 카드 |
| **CTA** | 4개 외부 채널 버튼 (밴드/인스타/몰/비움마켓) |
| **Footer** | 브랜드 소개, 링크 모음, 소셜 아이콘 |

## 파일 구조

```
index.html          ← 메인 랜딩페이지
css/
  style.css         ← 전체 스타일 (네이비/골드/틸 팔레트)
js/
  main.js           ← 인터랙션 (카운터, 카운트다운, 파티클, AOS)
README.md
```

## 주요 기능

- **숫자 카운터 애니메이션** — IntersectionObserver 기반, 화면에 보일 때 easeOut 카운팅
- **카운트다운 타이머** — 비움마켓 런칭 D-day 실시간 카운트다운
- **이메일 사전등록 폼** — 유효성 검사 포함
- **AOS 스크롤 애니메이션** — 각 섹션 페이드인/슬라이드 효과
- **파티클 배경** — 비움마켓 섹션 동적 파티클
- **히어로 패럴랙스** — 스크롤 시 배경 레이어 패럴랙스
- **카드 틸트 효과** — 마우스 무브 기반 3D 회전
- **반응형** — 모바일/태블릿/데스크탑 완전 대응
- **모바일 햄버거 메뉴** — 스무스 토글
